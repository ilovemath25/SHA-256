# SHA256 Hash Generator

A simple web-based SHA256 hash generator from scratch.

## Features

- Generate SHA256 hash from input text.
- Customize output: uppercase, spaced hex, or binary.
- Responsive design.

## Usage

1. Open `index.html` in a browser.
2. Enter text in the "Input Text" area.
3. View the SHA256 hash in the "SHA256 Output" area.
4. Use switches to customize the output.
